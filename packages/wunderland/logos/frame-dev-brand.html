Usage Guidelines

Minimum Size

Primary logo: 140px width
Icon: 16px minimum

Clear Space

Maintain clear space equal to the height of the icon frame around the logo

Color Usage

Use monochrome variations to match your brand context. Never modify the colors.

Documentation

Use black or blue variant for technical documentation and developer resources

Marketing

Use colored variants (green, purple, coral) for marketing materials and social media

Enterprise

Use navy or black variant for enterprise and corporate communications
