# Upload Instructions for .github Profile Repo

## What to do:

1. Go to your `.github` repo: https://github.com/framersai/.github

2. Create a `profile/` folder if it doesn't exist

3. Upload these 4 SVG files to the `profile/` folder:
   - `frame-logo-with-tagline.svg`
   - `frame-logo-no-subtitle.svg`
   - `agentos-icon.svg`
   - `openstrand-logo-gradient.svg`

4. Copy the updated README content from `FRAMERS_ORG_README.md` to the `.github` repo's `profile/README.md`

## File Locations:

All logos are in: `.github-profile-assets/`

The updated README is in: `FRAMERS_ORG_README.md`

## Result:

Once uploaded, the README will display properly with all working logo images at:
https://github.com/framersai

The logos will be accessible at URLs like:

- `https://raw.githubusercontent.com/framersai/.github/master/profile/frame-logo-with-tagline.svg`
- `https://raw.githubusercontent.com/framersai/.github/master/profile/agentos-icon.svg`
- etc.
