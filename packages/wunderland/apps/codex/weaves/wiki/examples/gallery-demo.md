# Art Deco Gallery Demo

This strand demonstrates the automatic gallery generation for multiple images.

## Individual Images

Single images display normally:

![A beautiful sunset](https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=800)

## Automatic Gallery Wall

When you have 3 or more images in sequence, they transform into a stunning Art Deco gallery:

![Mountain landscape](https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=600)
![Ocean waves](https://images.unsplash.com/photo-1505142468610-359e7d316be0?w=600)
![Forest path](https://images.unsplash.com/photo-1441974231531-c6227db76b6e?w=600)
![Desert dunes](https://images.unsplash.com/photo-1509316785289-025f5b846b35?w=600)
![City lights](https://images.unsplash.com/photo-1514565131-fce0801e5785?w=600)

## Features

- **Golden Ratio Layout**: Images are sized using Ï† (1.618)
- **Art Deco Frames**: Each image gets an ornate golden frame
- **Hover Effects**: Images scale up on hover
- **Responsive**: Adapts to mobile screens
- **Theme Aware**: Works with light, dark, and sepia themes
- **Lightbox Ready**: Click to view full size (coming soon)
