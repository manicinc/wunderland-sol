Usage Guidelines

Minimum Size

Primary logo: 120px width
Icon: 16px minimum

Clear Space

Maintain padding equal to the height of the "Q" around all sides

Light Backgrounds

Use Green or Mono Light variants on white/light gray backgrounds

Dark Backgrounds

Use Red or Mono Dark variants on black/dark backgrounds

Social Media

Use 512×512 icon for profiles. Full logo for banners and covers

Favicon

Use simplified icon at 16×16 and 32×32 for browser tabs
