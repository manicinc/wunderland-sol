# Keyboard Shortcuts

Master Quarry with these keyboard shortcuts. Press `?` or `k` to view shortcuts in-app.

## Quick Reference

| Shortcut | Action |
|----------|--------|
| `/` | Focus search |
| `?` | Show help panel |
| `k` | Show keyboard shortcuts |
| `,` | Open settings |
| `Esc` | Close modal/panel |

## Navigation

| Shortcut | Action |
|----------|--------|
| `/` | Focus search |
| `↑/↓` or `j/k` | Navigate up/down |
| `←/→` or `h/l` | Toggle sidebar/metadata |
| `w/s` | Navigate (WASD mode) |
| `Shift+↑/↓` | Scroll up/down |
| `Page Up/Down` | Fast scroll |
| `Home/End` | Jump to top/bottom |
| `gg` | Jump to top (Vim mode) |
| `Enter/Space` | Activate/select item |
| `g h` | Navigate home |
| `s` | Toggle sidebar (mobile) |
| `b` | Toggle bookmarks panel |
| `t` | Open today's daily note |

## Vim-Style Navigation

Quarry supports Vim-style navigation for power users:

| Shortcut | Action |
|----------|--------|
| `j` | Move down |
| `k` | Move up |
| `h` | Move left / collapse |
| `l` | Move right / expand |
| `gg` | Go to top |
| `G` | Go to bottom |

## Tips

1. **Search Everywhere**: Press `/` from anywhere to start searching
2. **Quick Create**: `⌘/Ctrl+N` creates a new strand instantly
3. **Vim Mode**: Use `j/k` for navigation if you're a Vim user
4. **Context Shortcuts**: Different views have context-specific shortcuts
5. **Customize**: Check Settings → Shortcuts to customize bindings
