# Sample Note

This is an example strand that demonstrates the standard format for Quarry documents.

## Basic Formatting

**Bold text** and *italic text* work as expected.

You can also use `inline code` and create code blocks:

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}
```

## Lists

### Ordered
1. First step

3. Third step

### Task Lists
- [x] Completed task
- [ ] Pending task
- [ ] Another pending task
