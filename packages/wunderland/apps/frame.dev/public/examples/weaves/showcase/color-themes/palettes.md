## Dark Theme Variations

For dark theme compatibility, use these darker backgrounds:

```yaml
# Dark blue theme
style:
  accentColor: "#60a5fa"
  backgroundColor: "#1e3a5f"
  textColor: "#e0f2fe"
  darkText: false
```
